## Introduction

**LLMR** is an open-source application designed to seamlessly run and manage Large Language Models (LLMs) with multiple settings, including popular ChatGPT variants from OpenAI. Whether you're a researcher, educator, or student, LLMR provides a user-friendly platform that requires no coding. All data and configurations are stored locally on your device, ensuring privacy and security.

---

## Features

- **Run Multiple LLMs:** Easily switch between various language models, including ChatGPT and other OpenAI variants.
- **No Coding Required:** User-friendly interface designed for users without programming experience.
- **Local Data Storage:** Ensures all data remains on your device, maintaining privacy and security.
- **Customizable Settings:** Adjust model parameters to suit your specific research or educational needs.
- **Cross-Platform Support:** Available on Windows, macOS (Apple Silicon and Intel), and Linux.

We welcome contributions to **LLMR**! Please follow these steps to contribute:

1. **Fork the Repository**
   - Click the "Fork" button at the top right of this repository to create your own copy.

2. **Clone Your Fork**
   ```bash
   git clone https://github.com/mmsbld/LLMR.git
   ```

3. **Create a Feature Branch**
   ```bash
   git checkout -b feature/YourFeature
   ```

4. **Commit Your Changes**
   ```bash
   git commit -m "Add Your Feature"
   ```

5. **Push to the Branch**
   ```bash
   git push origin feature/YourFeature
   ```

6. **Open a Pull Request**
   - Navigate to your fork on GitHub.
   - Click the "Compare & pull request" button.
   - Provide a clear description of your changes and submit the pull request.
